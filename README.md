# Ecomm

Repo for building ecommerce:

- **`synnex-shopify-sync/`** — AWS Lambda: TD Synnex → Shopify sync (SAM).
- **`storefront/`** — Next.js marketing site: carousel, category nav, discount email signup (Shopify Storefront API). See [storefront/README.md](storefront/README.md).
- **`lambda/`** — Legacy Synnex fetch sample.
